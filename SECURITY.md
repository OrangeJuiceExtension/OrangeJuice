# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Orange Juice, please report it responsibly.

**How to report:**
- Email: hello@oj-hn.com  
- Include:
  - A clear description of the issue
  - Steps to reproduce
  - Any relevant screenshots, logs, or proof-of-concept code
  - Potential impact, if known

**What to expect:**
- **Acknowledgment:** within 48 hours  
- **Initial assessment:** within 3–5 business days  
- **Regular updates:** as often as information changes

**Disclosure process:**
- If the vulnerability is accepted, we will work on a fix and coordinate disclosure with you.
- We ask that you do not publicly disclose the issue until a fix is released.
- Once resolved, we’re happy to credit you (if you want) in release notes or a security acknowledgments section.

**Out of scope:**
- Issues that do not impact security (e.g., general bugs, UI issues)
- Reports without sufficient detail to reproduce
- Automated or low-quality bulk reports

We appreciate responsible disclosure and the time you take to help improve the security of Orange Juice.
